# KORION Wallet Holo Card

Standalone React + TypeScript + Vite + Capacitor demo for the KORION NFT holo card component.

## Run

```powershell
cd C:\Users\jyk86\Documents\KORION-WALLET-HOLO-CARD
npm install
npm run dev
```

Open:

```text
http://127.0.0.1:5173
```
